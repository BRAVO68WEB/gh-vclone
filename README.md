# gh vclone - GitHub Visual Repo Clone

## Description 💭

This is a simple script that allows you to clone a GitHub Organization or User's repositories in a visual way.

## Installation 🛠️

```bash
gh extension install bravo68web/gh-vclone
```

## Usage 📖

```bash
gh vclone <user/org>
```
